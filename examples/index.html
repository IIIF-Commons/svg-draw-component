svg-draw-component

The svg-draw-component allows a user to draw shapes on a HTML5 canvas. Users have the option to overlay the canvas on images, an OpenSeadragon instance, or a plain canvas (default). To try out the component on your own, there are three basic steps:

  1. Include the bundled svg-draw-component and base-component files (along with OpenSeadragon distribution files if using it)
  2. Add a <div id="svgdraw"> element in the body
  3. Add the following below your svgdraw div 
                <script type="text/javascript">

                      var svgdraw;

                      $(function() {

                          svgdraw = new IIIFComponents.SvgDrawComponent({
                              element: "#svgdraw",
                              overlayType: "" // other options are "img" or "osd"
                          });

                      });
            </script>
